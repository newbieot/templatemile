📌 PANDUAN PENGGUNAAN (SOP)

  1. Capture & Convert: Foto seluruh kiriman (paket/dokumen), jadikan 1 file PDF (Gunakan CamScanner di HP).
  2. Ekstrak Data: Buka gemini.google.com (akun Gemini Pro), upload PDF, dan ketik: "Buatkan tabel nama penerima, alamat penerima, no surat (jika ada), dan no hp penerima, lalu convert ke .xlsx".
  3. Upload Mentahan: Drag & Drop file hasil .xlsx (atau file klien) ke area Upload Berkas di bawah. Sistem pintar akan melacak data meskipun Sheetnya acak!
  4. Lengkapi Data: Isi ID Pelanggan, Kode Layanan, dll.
  5. Review & Edit: Cek tabel di kanan. Kodepos otomatis diisi AI dengan membaca kata paling belakang, tapi bisa diedit manual.
  6. Selesai: Klik Export. File Excel valid siap di-upload ke Mile App tanpa entri manual.
