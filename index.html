📌 PANDUAN PENGGUNAAN (SOP)

  1. Capture & Convert: Foto seluruh resi/alamat, jadikan 1 file PDF (via CamScanner di HP).
  2. Ekstrak Data: Buka gemini.google.com, upload PDF, dan ketik: "Buatkan tabel nama penerima, alamat penerima, no surat (jika ada), dan no hp penerima, lalu convert ke .xlsx".
  3. Upload Mentahan: Drag & Drop file .xlsx ke area Upload Berkas di bawah. Sistem pintar melacak data secara otomatis.
  4. Lengkapi Data: Pilih mode (Korporat/Ritel) dan isi form pengirim.
  5. Selesai: Klik Export. File Excel valid siap di-upload ke Mile App tanpa entri manual.
